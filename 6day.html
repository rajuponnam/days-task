html
  • HTML stand for hyper text markup language
  • It was introduced in 1991 by tem berners lee
  • In HTML a markup language is used to structure and present content on the web by using tags and attributes to define element such as heading, links, images,and other multimedia.
heading tags
HTML headings are defined with the h1 to h6 tags.h1 defines the most important heading. h6 defines the least important heading
paragraph tag
The HTML p element defines a paragraph.A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph
formatting tags
  1. Bold tag : Make the text bold without adding any extra importance
  2. Italic tag : Makes the text italic without adding any extra emphasis
  3. Strong tag :The HTML strong element defines text with strong importance. The content inside is typically displayed in bold.
  4. Emphasize tag :The HTML em element defines emphasized text.
  5. Underline tag : underline the text
  6. Subscript :The HTML sub element defines subscript text. Subscript text appears half a character below the normal line
  7. Superscript :The HTML sup element defines superscript text. Superscript text appears half a character above the normal line,
  8. Deleted tag :The HTML del element defines text that has been deleted from a document.
  9. Mark tag :The HTML mark element defines text that should be marked or highlighted
  10. Blockquote tag :The HTML blockquote element defines a section that is quoted from another source.
  11. Abbreviation tag :The HTML abbr tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".
  12. Quatation tag :Browsers normally insert quotation marks around the quotation.
  13. Cite tag :The HTML cite tag defines the title of a creative work
  14. Address tag :The HTML address tag defines the contact information for the author/owner of a document or an article
  15. Code tag :Define some text as computer code in a document
  16. Keyboard tag :Define some text as keyboard input in a document
Blockline element
A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
Inline element
An inline element does not start on a new line.
Table tag
  1. It is used to represent content in a tabular form
  2. Table are defined with table tag
  3. Rows are represented with tr
  4. columns are represented with td
  5. columns headings are represented with th
Table features
  • Table caption
  • Cell padding
  • Cell spacing
  • Background color
  • Width and Height

images tag
The img tag is used to embed an image in an HTML page
Anchor tag
The a tag defines a hyperlink, which is used to link from one page to another.
List tag
The ul(unordered list), ol(ordered list),dl(description list) and used to creates a bulleted list, the ol tag creates a numbered list, and the li tag define individual list items within both types of lists.
